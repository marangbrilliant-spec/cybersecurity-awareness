# Cybersecurity Awareness Mini Project  
**Module:** C5-ICS-23 Cybersecurity Awareness  
**Author:** Marang Brilliant Gaogalalwe-2540805

## Overview  
This repository contains a mini project focused on **cybersecurity awareness**, created as part of the BSc Cybersecurity and Risk Management programme. It includes documentation, scripts, and reports designed to promote safe online practices and demonstrate practical cybersecurity knowledge.

## Contents  
- **research_notes.md** – Key cybersecurity threats and mitigation strategies  
- **password_strength_checker.py** – A Python script to test password strength  
- **incident_response_template.docx** – A professional response template for cyber incidents  
- **threat_report_summary.md** – A short report summarising common attacks and defences  

## Purpose  
The project aims to enhance understanding of:
- Cyber threats and risks  
- Safe password creation  
- Organised incident response  
- The use of GitHub for professional documentation and collaboration
